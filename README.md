# 🍽️ Restaurant Management System

A desktop-based restaurant management application designed to streamline restaurant operations such as order processing, billing, and inventory management.

## 🚀 Features

- 🛒 Order Management  
  Create, update, and manage customer orders efficiently.

- 💳 Billing System  
  Generate bills automatically with accurate calculations.

- 📦 Inventory Management  
  Track stock availability and manage restaurant inventory.

- 🖥️ User-Friendly GUI  
  Interactive desktop interface built using Tkinter.

- 🗄️ Database Integration  
  MySQL database for secure and structured data storage.

---

## 🛠️ Tech Stack

- **Programming Language:** Python
- **GUI Framework:** Tkinter
- **Database:** MySQL

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/pratik26x/Restaurant-Management-System.git
cd Restaurant-Management-System
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

If requirements file is not available:

```bash
pip install mysql-connector-python
```

---

### 3. Setup MySQL Database

- Open MySQL
- Create a database:

```sql
CREATE DATABASE restaurant_management;
```

- Update database credentials in your Python project:

```python
host="localhost"
user="your_username"
password="your_password"
database="restaurant_management"
```

---

### 4. Run the Project

```bash
python main.py
```
